Welcome to CauseDB

CauseDB is a database that uses a combination of SQL and JSON. It is easy to use, making it a great choice for those who need a simple database solution.

Features

  • MYSQL it is used to as commands to the database.

  • JSON it is used to create and manage data.

  • Dashboard a way to manage the database without a driver

  • Drivers Create your own driver or find a driver.

  • Web Interface a way to manage the database without a driver using the dashboard.

  • Driverless a way to manage the database without a driver.

Getting Started

Step 1: To get started, you will need to create a database. This can be done by clicking on the 3 bars in the top left of the screen. and select the Create option. Step 2: Once you have created a database, you will need to create a table. This can be done by clicking on the 3 bars in the top left of the screen. and select the Create option and the select a database to create a table in. Step 3: Once you have created a table, you will need to create a row. This can be done by clicking on the 3 bars in the top left of the screen. and select the Create option and the select a database to create a and select a table to create a row in.

Our Story

When all other databases failed him, one little programmer took matters into his own hands and created CauseDB. ThatFriendlyAsianDev was working on a project when he ran into trouble finding a database that would work for him. He tried RethinkDB and Redis but kept getting errors. Finally, he couldn't take it anymore and decided to create his own database. And so CauseDB was born! CauseDB is a powerful and reliable database that has helped him to create a lot of projects.
