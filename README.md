## Installation
To Install this project, follow the steps above:
```bash
# install

npm install causedb -g

```

## Usage
To use this project, follow the steps above:
```bash
# Running

cause --help

#start the server

cause start --host localhost --port 80 --dashboardusername admin --dashboardpassword test123
```
